# Planos de execução

Planos detalhados por objetivo, escritos sob demanda — um plano por objetivo de fase, com passos, arquivos tocados, testes e critério de pronto.

Convenção de nome: `faseN-<objetivo>.md` (ex: `fase1-vault-fs.md`, `fase1-parser.md`, `fase1-frontmatter.md`).

> Vazio por enquanto. Os planos são criados quando começamos a executar cada objetivo. Visão geral das fases em [`../fases/`](../fases/).
